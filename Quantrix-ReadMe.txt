The contents of this project are the forked JFreeChart project used by Quantrix (subx, inc). 
The project was forked due to specific requirements to integrate JFreeChart with Quantrix charting and to fix bugs more quickly.
Some of these changes were submitted back to the original project.
The files that were changed by Quantrix are marked in the source header with Quantrix Modifications followed by the dates of change,
by whom, and what the purpose of the change was.  Changed code is often marked by the developer.

A complete listing of the history of this project is found in QuantrixJFreeChartChanges.txt.
